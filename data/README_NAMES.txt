
AD-SIM Name Shards
------------------
This folder contains sharded name lists suitable for browser loading.
Distribution: ~60% US/AU/Western Europe, ~40% Asia/Africa/Nordics/Other Europe.
Totals in this drop:
  Male first names  : 300
  Female first names: 300
  Last names        : 600

Files are referenced by data/names-index.json (version 2).
Add more shards by creating files like first-male-004.json and appending the filename
into the corresponding array in names-index.json.
